{{include.para}} custodian

See reference for details on this parameter. The custodian parameter can be used as follows:

GET [baseUrl]/{{include.content}}?custodian.identifier=[system]|[code]

system is a uniform resource identifier which defines which CodeSystem the identifer belongs to. For ODS this would be https://fhir.nhs.uk/Id/ods-organization-code and code would be the ODS identifier.

GET [baseUrl]/{{include.content}}?custodian.identifier=https://fhir.nhs.uk/Id/ods-organization-code|RR8

Return all {{include.content}} resources for pointer owners (Custodians) with an ODS code of RR8
