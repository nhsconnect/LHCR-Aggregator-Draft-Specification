Note: If the `_count` parameter is included then the resulting Bundle that is returned may include page navigation links where appropriate. If a client issues a search request without the `_count` parameter and the number of records in the result-set exceeds the default page size (to be determined) then the resulting Bundle is also expected to use page navigation links to break up the result-set.
